Vane Data helps teams build pipelines for multimodal data preparation, AI-ready datasets, and batch inference with DuckDB-compatible SQL, Python/Arrow UDFs, Ray-backed execution, and AI helper functions.

Use it when documents, images, audio, video, text records, or model outputs need SQL filtering plus Python libraries, embeddings, prompting, model inference, GPU execution, or distributed reprocessing before they move to training, search, analytics, or downstream applications.

SQL is a first-class interface through `con.sql(...)` and `con.execute(...)`; Python is available through UDFs and AI helpers when the pipeline needs custom logic.

Install `vane-ai` and import `vane`.

Vane Data is a data processing layer for AI and multimodal workflows, not a model server, vector database, workflow scheduler, transactional database, or traditional structured ETL replacement.
