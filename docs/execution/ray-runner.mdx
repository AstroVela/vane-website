export const title = 'Ray Runner'

Each pipeline stage becomes Ray work. Stateless transforms map to tasks; stateful UDFs (models, tokenizers) map to long-lived actors so weights load once.

- SQL / filters → Ray tasks over Arrow blocks
- map_batches(fn) → tasks, or actors when fn is a class
- GPU models → model actors with num_gpus reservation
- Output → streamed writes to Parquet / S3

<Callout label="Note">

Vane streams blocks through the graph with backpressure, so memory stays bounded even on datasets larger than cluster RAM.

</Callout>
