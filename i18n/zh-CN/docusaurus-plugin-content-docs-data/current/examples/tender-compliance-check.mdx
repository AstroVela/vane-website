---
title: 招标合规检查
---

本示例描述使用 Vane Data 进行招标或采购合规检查的模式。

Vane Data 不内置招标专用解析器或合规引擎。它提供 SQL、relation API 和 AI helper，可组合成审查 workflow。

## 目标

将招标文件与检查清单对比，并产出可审查、抽样和审计的结构化合规结果。

## 输入表

文档：

- `document_id`
- `section`
- `text`
- `source_uri`

检查清单：

- `rule_id`
- `requirement`
- `severity`

## SQL 准备

```python
import vane

con = vane.connect()

docs = con.sql("""
    select document_id, section, text, source_uri
    from read_parquet('data/tender_docs/*.parquet')
    where text is not null
""")

rules = con.sql("""
    select rule_id, requirement, severity
    from read_csv_auto('data/tender_rules.csv')
""")
```

## 构建检查项

使用 SQL 让模型工作负载显式化。在将提示词发送给 provider 前，先检查交叉连接规模。

```python
docs.to_table("docs")
rules.to_table("rules")

checks = con.sql("""
    select
        d.document_id,
        d.section,
        d.source_uri,
        r.rule_id,
        r.severity,
        'Requirement: ' || r.requirement || '\nDocument text: ' || d.text as prompt
    from docs d
    cross join rules r
""")
```

## 结构化模型检查

`prompt(...)` 返回已配置的输出列。汇总前，请显式地将结果与检查行组合，并在写出结果前校验行数。

```python
from pydantic import BaseModel

class ComplianceResult(BaseModel):
    compliant: bool
    evidence: str
    reason: str

result_only = checks.prompt(
    "prompt",
    provider="openai",
    return_format=ComplianceResult,
    output_column="result_json",
    execution_backend="subprocess_actor",
)

checks_table = checks.to_arrow_table()
result_table = result_only.to_arrow_table()
checked = con.from_arrow(checks_table.append_column("result_json", result_table["result_json"]))
```

## 汇总

```python
checked.to_table("checked")

summary = con.sql("""
    select
        document_id,
        severity,
        count(*) as checks
    from checked
    group by document_id, severity
""")

summary.show()
```

## 扩展

当 SQL 准备、UDF 阶段或模型调用需要分布式 worker 时，使用 Ray：

```python
import vane

vane.configure(runner="ray")
```

根据 API 限制、速率限制和模型服务容量设置 provider 并发度。

## 校验

在合规 workflow 中使用结果前：

- 对检查清单和提示词进行版本化。
- 保留源文档引用。
- 人工审查高严重级别失败项。
- 抽样检查合规和不合规输出。
- 将模型输出视为审查证据，而不是法律建议。

## 范围说明

本页不定义法律建议、采购政策、文档解析或招标专用连接器。它记录如何使用 Vane Data SQL 和 AI helper 表达该 workflow。
