Vane Data 帮助团队构建面向多模态数据准备、适合 AI 的数据集和 batch 推理的处理 pipeline。它提供兼容 DuckDB 的 SQL、Python/Arrow UDF、基于 Ray 的分布式执行，以及面向 AI 工作负载的辅助函数。

当文档、图像、音频、视频、文本记录或模型输出在进入训练、搜索、分析或下游应用之前，需要 SQL 过滤、Python 库处理、embedding 生成、提示词调用、模型推理、GPU 工作或分布式重处理时，可以使用 Vane Data。

SQL 通过 `con.sql(...)` 和 `con.execute(...)` 作为一等接口提供；当 pipeline 需要自定义逻辑时，可以通过 UDF 和 AI 辅助函数接入 Python。

安装 `vane-ai` 并导入 `vane`。

Vane Data 是面向 AI 和多模态 workflow 的数据处理层，不是模型服务器、向量数据库、workflow 调度器、事务数据库，也不是传统结构化 ETL 的替代品。
