export const title = 'AI Functions'

Built-in helpers wrap common models as ready-to-use batch UDFs. They run as model actors and honor num_gpus and concurrency.

| Function | Does |
|---|---|
| embed | text / image embeddings |
| generate | LLM completion (vLLM) |
| caption / vlm | image+text → structured output |
| transcribe | audio → text |
| judge | grade outputs against a rubric |
