export const title = 'SQL / Relation API'

Vane is DuckDB-compatible: write SQL, get a lazy Relation you can chain Python transforms onto. Nothing executes until you materialize or write it.

| Method | Returns |
|---|---|
| .map_batches(fn) | Relation |
| .flat_map(fn) | Relation |
| .filter(expr) | Relation |
| .write_parquet(path) | None (executes) |
| .to_arrow() | pyarrow.Table |
